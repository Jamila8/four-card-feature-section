# Frontend Mentor - Four card feature section solution

This is a solution to the [Four card feature section challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/four-card-feature-section-we61NsbS-). 

### The challenge
Users should be able to:
- View the optimal layout for the site depending on their device's screen size.
- See the hover states for all interactive elements.

### My Process
- **HTML5:** Semantic markup for structure.
- **CSS Grid:** Used for the multi-column, tiered card layout.
- **CSS Flexbox:** Used for internal card alignment and icon positioning.
- **Mobile-first workflow:** Ensuring responsiveness.

### What I learned
In this project, I practiced advanced **CSS Grid** positioning, specifically using `grid-row: span 2` to create the featured center column effect. I also improved my skills in aligning icons within cards using `margin-left: auto`.
